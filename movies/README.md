# Spring Boot Movie Backend

This is a Spring Boot backend application that provides REST APIs for managing movies.  
It is designed to work with a React frontend to create a fullstack movie application.

## 🚀 Features
- RESTful APIs for movie CRUD operations
- Integration with a relational database (e.g., MySQL/PostgreSQL)
- Spring Data JPA for ORM
- Exception handling and validation
- Ready to extend for authentication and user management

## 🛠️ Tech Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- SQL Database (MySQL/PostgreSQL)
- Maven
